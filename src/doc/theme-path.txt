wp-theme-path(1) -- Get the path to a theme or to the theme directory.
====

## SYNOPSIS

`wp theme path` [<theme>] [--dir]

## OPTIONS

* `<theme>`:

	The theme to get the path to. If not set, will return the path to the
themes directory.

* `--dir`:

	If set, get the path to the closest parent directory, instead of the
theme file.
