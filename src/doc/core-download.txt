wp-core-download(1) -- Download core WordPress files.
====

## SYNOPSIS

`wp core download` [--locale=<locale>] [--version=<version>]

## OPTIONS

* `--locale`=<locale>:

	Select which language you want to download. The --version parameter is
ignored in this case.

* `--version`=<version>:

	Select which version you want to download.

## EXAMPLES

	Download version 3.3: wp core download --version=3.3
